# TechCare Assets

## Required Images

Place the following images in the respective folders:

### Logos (`assets/logos/`)
- `techcare_logo.png` - TechCare Medical Services logo (from Image 1)
- `hospital_logo.png` - Plaridel Community Hospital logo (from Image 2)

### Hero Section (`assets/images/`)
- `hero_1.png` - TechCare Medical Services logo with hands (from Image 1)
- `hero_2.png` - Plaridel Community Hospital logo (from Image 2)

### About Section (`assets/images/`)
- `hospital_1.png` - Hospital exterior front view (from Image 4)
- `hospital_2.png` - Hospital exterior side view (from Image 5)
- `hospital_3.png` - Hospital interior nurse station (from Image 6)

## Image Specifications

- **Format:** PNG with transparent background for logos
- **Hero images:** Recommended size 800x800px
- **About carousel:** Recommended size 700x500px
- **Logos:** Recommended size 200x200px

## Notes

The carousel will automatically handle different image sizes, but maintaining consistent aspect ratios will provide the best visual experience.
